# Contributing to "Co robisz? (rewrite)"

First off, thank you for considering contributing to "Co robisz? (rewrite)"! I welcome community involvement to make this game better.

## How to Contribute
1.  **Fork** the repository (for the purpose of creating a Pull Request only).
2.  Create a **branch** for your feature or fix.
3.  Commit your changes.
4.  Submit a **Pull Request (PR)** to the `main` branch.

## Recognition
If your Pull Request is accepted and merged:
* You will be added to the in-game **Contributors List**.
* You will have my eternal gratitude for helping build this project!
